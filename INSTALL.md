# 설치 가이드

이 프로젝트는 **각 패키지별로 독립 설치**를 기본 원칙으로 합니다.

## ⚠️ 중요 사항

**루트에서 `npm install`을 실행하지 마세요!**  
각 패키지 디렉토리에서 개별적으로 설치해야 합니다.

## 설치 순서

### 1단계: Common 패키지 (공통 라이브러리)

cd packages/common
npm install
cd ../..

```

### 2단계: Functions 패키지
cd packages/functions
npm install
cd ../..

```

### 3단계: Web CMS 패키지

cd packages/web-cms
npm install
cd ../..

```

### 4단계: Mobiles 패키지
cd packages/mobiles
npm install
cd ../..

```
